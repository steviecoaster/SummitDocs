Open-GitRepo

SYNOPSIS

Changes your $pwd to the directory of the requested repo name

SYNTAX

Open-GitRepo [-Repo] <String> [<CommonParameters>]

DESCRIPTION

Changes your $pwd to the directory of the requested repo name

EXAMPLES

EXAMPLE 1

Open-GitRepo -Repo SuperAwesomeProject

PARAMETERS

-Repo

The repo to open

Type: String
